# "Просмотреть публикацию"

### Краткое описание

Вариант использования "Просмотреть публикацию" позволяет любому пользователю платформы перейти к странице просмотра существующей публикации.

### Основной поток:
1. Вариант использования начинается, когда пользователь кликает по кнопке "Читать продолжение" малой карточке публикации.
2. В окне браузера выводится страница просмотра публикации
